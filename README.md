# Task Manager REST API

Simple REST API created with FastAPI, SQLite and SQLAlchemy.

## Features

- Create tasks
- Get all tasks
- Get one task by ID
- Update tasks
- Delete tasks
- SQLite database
- Swagger documentation

## Technologies

- Python
- FastAPI
- SQLAlchemy
- SQLite
- Pydantic
- Uvicorn
